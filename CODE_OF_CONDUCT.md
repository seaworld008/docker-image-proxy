# 社区行为准则

本项目希望保持专业、友善、可协作的开源环境。参与 Issue、PR、讨论和文档贡献时，请遵守以下原则。

## 我们鼓励

- 尊重不同经验水平和背景的贡献者。
- 用事实、日志、配置和复现步骤讨论问题。
- 对风险和限制保持诚实，不夸大功能或结论。
- 帮助新用户找到正确文档入口。
- 在安全问题中主动脱敏，避免公开泄露敏感信息。

## 不接受

- 人身攻击、侮辱、骚扰或歧视性表达。
- 故意提交真实密钥、token、私钥或他人隐私信息。
- 在公开 Issue 中披露可直接利用的安全细节。
- 刷屏、广告、与项目无关的推广内容。
- 明知未验证仍宣称兼容、生产可用或具备某项能力。

## 处理方式

维护者可以编辑或删除不合适内容，关闭偏离项目目标的 Issue/PR，或限制持续破坏协作氛围的账号参与。

如果你发现违反本准则的行为，请通过 GitHub Issue、GitHub Security Advisories 或维护者可用联系方式反馈。涉及安全或隐私的信息请优先使用私密渠道。
