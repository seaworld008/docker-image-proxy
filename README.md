# Docker 镜像加速自建方案

本仓库提供一份可直接落地的生产级文档，指导你自建 Docker Hub 镜像加速服务，包含两种方案：

- 方案 A：内网部署 + 机场订阅代理出海
- 方案 B：海外源站 + CDN 全局缓存加速

文档覆盖部署、配置、CDN 各厂商控制台设置、安全/鉴权、K8S 节点配置、测速与排障等内容。
